# Selectel_2_task
Автотесты для сайта Selectel.ru

# Какие проверки реализованы

Написано 4 базовых сценария для шапки сайта:

testPageTitle(): проверка, что в Title вкладки есть слово «Selectel».

testLogoIsDisplayed(): проверка, что главный логотип корректно отображается.

testDocsLinkOpensInNewTab(): проверка клика по ссылке, переключение на новую вкладку.

testProductsMenuOpens(): эмулируем клик по кнопке навигации и проверяем, что выпадающее меню появилось на экране.

# Какие инструменты использовала

Java

Selenium WebDriver 4 

JUnit 5 

Maven

# Как запустить проект локально

Для запуска понадобятся установленные Java (JDK), Maven и браузер Google Chrome.

чтобы запустить тесты, нужно выполнить команду
mvn clean test
